# Local Notes

一个可以直接在浏览器中打开的 Markdown 笔记工具，可选择同步到 GitHub 仓库里的 `.md` 文件。

## 使用

双击打开 `index.html`，或在浏览器里打开这个文件。

笔记默认保存在浏览器的本地存储中。填写 GitHub 同步设置和 token 后，可以把每条笔记同步成仓库 `obsidian-notes/` 目录下的 `.md` 文件。

注意：这个站点托管在公开的 GitHub Pages 仓库中，同步到仓库的 `.md` 笔记也是公开内容，不要写隐私信息。

## 功能

- Markdown 编辑和预览
- 搜索标题、正文和标签
- `#标签`
- `[[笔记名]]` 双向链接
- 反向链接
- 简单关系图
- JSON 导出和导入
- GitHub `.md` 文件拉取和推送
